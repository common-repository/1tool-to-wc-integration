Provides a seamless integration to sync products and create invoice on 1Tool with WooCommerce.

== Description ==

This plugin provides integration to sync products from 1tool to WooCommerce and when order is placed on WooCommerce, it will be sent to 1tool via API. It also generates Invoice for that order on 1tool and create that customer there.

This requires an account with 1Tool to get API Credentials from there to connect it with WooCommerce.

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/folder.
2. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

1. How do I use this plugin?   
   Install this plugin from plugin library and activate it.

2. How to configure the plugin?
   Go to Plugin's Settings page, and enter your App ID and App Secret Key you can get from your 1Tool account dashboard.

3. How to uninstall the plugin?
   Simply deactivate plugin and delete it.

4. How to get support if plugin doesn't work as expected?
   You can contact us here in support ticket and we will revert you back as soon as possible.
